# startup-diligence-copilot

A workspace for angels and micro-VCs to analyze startup documents, surface risk, ask evidence-backed questions, and draft investment memos with traceable citations.

## Features

- Upload pitch decks, data rooms, and startup documents
- AI-powered diligence Q&A
- Risk and inconsistency detection
- Public web enrichment and research
- Citation-backed answers
- Investment memo generation
- Founder follow-up question generation

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- OpenAI APIs
- Vector search / RAG pipeline

## Getting Started

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Vision

Build an AI copilot for early-stage investors that reduces diligence time while improving decision quality.
